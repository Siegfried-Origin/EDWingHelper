ED Wing Helper

A simple tool to organize large player gatherings in Elite Dangerous by automating wing invitation tracking.

Download Latest Release View on GitHub

Why ED Wing Helper?

Creating large instances in Elite Dangerous requires inviting commanders three at a time and tracking who has already joined. ED Wing Helper automates this tracking process so the wing leader can focus on organizing the event instead of managing lists manually.

Commander Lists

Import a simple text file containing commander names. One name per line is all you need.

Journal Monitoring

Automatically reads Elite Dangerous journals to detect friend activity and wing invites.

Online Tracking

Commanders are automatically sorted depending on whether they are online or offline.

Invitation Tracking

Accepted wing invites are automatically moved to the invited list.

Disconnect Detection

If an invited commander disconnects, they are moved back to the invite list.

List Management

Append, edit, and export commander lists easily.
